# Product: Debt Payoff Manager

A mobile app (iOS + Android) that helps users create and maintain a **living debt payoff plan** — one that adapts in real time as payments are logged, strategies change, or new debts are added.

## Core Value Proposition

- **Living Plan, not Static Calculator** — the debt-free date updates instantly when anything changes
- **Local-first trust** — no bank linking, no account required to start, data stays on device
- **Mathematical precision** — snowball/avalanche strategies with exact financial calculations
- **Never touch your money** — planning and tracking only

## Feature Tiers

- **Tier 1 (MVP):** Debt tracking, snowball/avalanche strategies, payment logging, monthly action view, living timeline, guided onboarding
- **Tier 2 (Post-MVP):** What-if scenarios, milestones, partner sharing, notifications, PDF/CSV export

## Progressive Trust Model

- Level 0: Local only (Drift/SQLite), no account
- Level 1: Optional Firestore backup (opt-in)
- Level 2: Firestore + household sharing

## Current Status

Financial engine and data layer are implemented and tested. Feature UI is in progress.
